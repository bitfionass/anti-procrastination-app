# 和拖延say拜拜 👋 — Web版

## 快速启动

```bash
# 方式1: Python HTTP服务器
cd app
python3 -m http.server 8081
# 打开 http://localhost:8081

# 方式2: Node.js (如果安装了)
npx http-server -p 8081

# 方式3: 直接双击 index.html (部分功能可能受限)
```

## 功能说明

### 🤖 AI任务拆解
- 输入模糊任务（如"写论文"），AI自动拆解为具体步骤
- 支持5个快捷模板：论文、复习、小组作业、学技能、整理
- 3档难度：简单、中等、困难
- 步骤可编辑、增删

### ⚡ 5分钟启动器
- 点击"先做5分钟"，开始倒计时
- 专注做第一步，降低启动心理门槛
- 倒计时结束自动标记完成

### 📊 进度反馈
- 实时进度条
- 完成任务获得勋章
- 统计：总完成数、连续天数、今日完成

### 🐾 电子宠物
- 完成任务喂养宠物，获得成长值
- 宠物有心情系统（不喂会不开心~）
- 5个等级：小萌新→行动派→效率达人→自律王者→传奇人物

## 数据存储
- 所有数据存在浏览器 LocalStorage
- 清除浏览器数据会丢失进度
- 后续版本将支持云端同步

## 技术栈
- HTML5 + CSS3 + 原生JavaScript
- 零框架依赖，可直接部署
- 响应式设计，手机优先
